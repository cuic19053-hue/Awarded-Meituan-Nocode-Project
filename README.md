# 智能会议室管理系统 - 项目概述

## 📋 项目简介

**智能会议室管理系统**是一个现代化的、基于Web的企业级会议室预订与管理平台。系统通过整合会议室预订、审批流程、能源监控和数据分析等功能，帮助企业实现会议室资源的科学化管理，提高使用效率，降低运营成本。

## 🎯 核心目标

1. **提高会议室使用率**：通过智能推荐和优化调度，最大化会议室资源利用率
2. **简化管理流程**：自动化预订、审批和管理流程，减少人工干预
3. **降低运营成本**：通过能耗监控和优化建议，降低能源消耗成本
4. **提供数据支持**：通过全面的数据分析，为企业决策提供数据支持

## 🏗️ 技术架构

### 前端技术栈
- **框架**: React 18 + Vite
- **样式**: Tailwind CSS + Radix UI
- **图表**: Recharts 数据可视化
- **表单**: React Hook Form + Zod
- **路由**: React Router DOM
- **图标**: Lucide React
- **动画**: Framer Motion

### 核心特性
- **模块化架构**: 组件化设计，易于维护和扩展
- **响应式设计**: 支持桌面端和移动端访问
- **实时更新**: 基于React状态管理的实时数据更新
- **离线支持**: 使用LocalStorage提供离线数据支持
